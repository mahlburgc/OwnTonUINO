Einschlafmodus
--------------
Um den Einschlafmodus zu aktivieren, muss die Pausetaste lange gedrückt werden.
Ist der Einschlafmodus aktiviert, wird die Beleuchtung gedimmt und die Musik und Beleuchtung geht nach einer bestimmten Zeit aus.
Die Zeit kann im Admin Menü eingestellt werden.


Schlüsselkarte
--------------
Die Schlüsselkarte ermöglicht das Sperren und Entsperren der Box und öffnet das Admin Menü.


Admin Menü
----------
Zum Öffnen des Admin Menüs siehe Tastenbelegung. Um das Adminmenü freizuschalten, muss ein Passcode eingegeben oder die Schlüsselkarte aufgelegt werden.
  Passcode (3 Tasten): links, mitte, rechts, rechts, mitte, links
  Passcode (5 Tasten): links unten, mitte, links oben, links oben, mitte, links unten

Mit den Lautstärketasten kann man durch die einzelnen Menüoptionen navigieren. Mit der Pausetaste wird die ausgewählte Option ausgeführt.
Ein langer Druck auf die Pausetaste beendet das Admin Menü.

Optionen im Admin Menü:
 1. Einschlaftimer konfigurieren - Es wird nur die Zeit neu konfiguriert, zum Aktivieren des Einschlafmodus siehe Abschnitt "Einschlafmodus"!
 2. Eine Karte neu konfigurieren - Gilt für bereits konfigurierte und unkonfigurierte Karten / Figuren!
 3. Beleuchtung ein-/Ausschalten - Die Beleuchtung zur Anzeige der Tastensperre und des Einschlafmodus bleiben immer aktiv!
 4. Maximale Lautstärker einstellen 
 5. Minimale Lautstärker einstellen
 6. Lautstärke bei Start einstellen
 7. Alle Einstellungen zurücksetzen

Die Einstellungen werden beim Ausschalten des Gerätes gespeichert.


Figuren oder Karten erstellen:
------------------------------
Es wird eine NFC Karte oder NFC Sticker (Mifare Classic) benötigt. Legt man die unkonfigurierte Figur/Karte auf, öffnet sich ein Stimmmenü und leitet durch die Konfiguration.
Zunächst muss der Abspielmodes ausgewählt werden (Verwendung von Lautstärketasten und Pausetaste).
 1. Albummodus     - alle Lieder nacheinander abspielen
 2. Partymodus     - alle Lieder in zufälliger Reihenfolge abspielen
 3. Hörbuchmodus   - alle Lieder nacheinander abspielen und den Fortschritt speichern
 4. Schlüsselkarte - siehe Abschnitt "Schlüsselkarte"
Danach wählt man den Ordner auf der SD-Karte aus, der mit der Figur abgespielt werden soll.

Möchte man eine bereits konfigurierte Figur/Karte neu konfigurieren, kann man das im Adminmenü unter Option "2. Eine Karte neu konfigurieren" machen.

Möchte man neue Musik auf den Tonuino spielen, muss die Musik auf die interne SD-Karte kopiert werden. Dazu muss die Box aufgeschraubt und die SD-Karte herausgenommen werden.
Die SD-Karte hat eine feste Ordnerstruktur. Diese Struktur muss eingehalten werden, damit die Musik ordentlich erkannt wird.
Es können insgesamt 99 eigene Ordner auf der SD-Karte angelegt werden. In jedem Ordner können bis zu 255 Lieder enthalten sein.

Benamung Ordner:  01 ...  99
Benamung Lieder: 001 ... 255
